Introduction

Apparu avec Internet, le développement web fait référence au processus d’écriture d’un site ou d’une page web dans un langage technique. Il s’agit d’une étape incontournable pour qu’un contenu soit mis en ligne et atteigne ses lecteurs.

Les types des sites web

Les web statique

On entend par page statique, non pas une page sans mouvements ou sans animations, mais une page visible telle qu'elle a été conçue. Ces pages peuvent présenter toute forme de contenu, animations flash, images, musique, vidéo etc...mais elles sont toujours présentées de la même façon. Elles ne changent pas et c'est en ce sens qu'elles sont statiques.

Les Web Dynamique

En opposition aux sites statiques, les sites dynamiques permettent de présenter les informations de différentes manières selon l'interaction avec le visiteur. Le contenu est issu d'une base de données en fonction de critères établis par l'internaute puis mis en page en temps réel.C'est le cas par exemple d'un site e-commerce: présentation des articles par thèmes, couleurs, prix etc...C'est également le cas des blogs et des forums où les visiteurs peuvent participer au contenu du site.

Les Web Dynamique

En contre partie , La réalisation d'un site web dyanmique fait appel aux langages de programmation de base (HTML , CSS et JS) et une autre langages avec lequelle développeur va programmer le fonctionnement du site web et établis les connexion et les relations entre la base de donnés du site web et le web lui-même. Le développement des relations entre le web et son base de donnés est appelé BackEnd. Autre que ça, le développeur de BackEnd doit aussi étudié une la langages de programmation de base de donnés (qu’on l’appelle SQL ou squel ou Structured Query Language) Mise a part ces deux types de développeur web (FrontEnd et BackEnd), on trouve aussi le développeur de FullStack (Full Stack Developer ) . Ce dérnier est capables de crée un site web tout seul et il est peut programmer tout. La difference entre lui et le BackEnd développeur est que le BackEnd développeur ne connaît pas comment utiliser les langages de base. Pour programmer la base de donnés on peur utilise : mysql, postgresql, orcale, micrsoft sql server…. Pour la programmation du BackEnd on peut utiliser : python(django ou flask ) , java (spring , Play, Tomcat, Jetty), PHP, node.js, Ruby …

Comment est le travail d’un développeur web

Les développeurs web occupent un poste sédentaire avec des horaires de travail classiques. Selon les délais de livraison, il peut être amené à élargir ses horaires mais de manière ponctuelle. Il peut exercer en entreprise, en SSIISociete de Services et d'Ingenierie Informatique (, en collectivité territoriale, en freelance).

Les Spécificités de travail

Un développeur web doit avoir la compréhension des contraintes du projet tel que le délai de réalisation du projet, le budget attribué aux tâches ainsi que les fonctionnalités attendues. Si les outils et logiciels utilisés par le développeur web ne cessent d’évoluer, ils deviennent aussi de plus en plus complexes. Tenir ses connaissances à l’affut de ces améliorations constantes est alors nécessaire pour le développeur web. Il se doit aussi de mettre à jour ses méthodes de travail, se conformer aux normes, connaitre les procédures de sécurité et d’outils de développement. Il doit être aussi à l’aise avec les langages de programmation tels que JAVA, PYTHON, Net, PHP, C++, …. Il doit aussi maitriser les autres langages informatiques comme le triplet HTML/CSS/Javascript qui est utilisé par les serveurs d’application ou les services web. Dans l’univers du multimédia, bien connaitre ou maitriser la langue anglaise est aussi indispensabl par le fait que la plupart des programmes utilisent cette langue
